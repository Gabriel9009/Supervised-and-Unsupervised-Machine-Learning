# 📊 Life Expectancy Prediction & Clustering Analysis

A machine learning project that applies both supervised and unsupervised learning techniques to analyze factors affecting life expectancy across different countries.

---

## 🚀 Overview

This project explores a global life expectancy dataset using data analysis, visualization, supervised machine learning models, and unsupervised learning techniques.

The goal of the project is to:
- Predict life expectancy using supervised learning models
- Discover hidden patterns and country groupings using clustering techniques
- Analyze how socioeconomic and health-related factors influence life expectancy

This project demonstrates the complete machine learning workflow from preprocessing and exploratory data analysis to model evaluation and clustering analysis.

---

- 📈 Exploratory Data Analysis (EDA)

## 🧠 Machine Learning Approaches

### ✅ Supervised Learning

Used for predicting life expectancy values.

Models explored include:
- Linear Regression
- Polynomial Regression
- Ridge Regression


### ✅ Unsupervised Learning

Used for identifying patterns and grouping countries with similar characteristics.

Algorithms explored include:
- K-Means Clustering
- Hierarchical Clustering (if applicable)

## 📊 Model Evaluation Metrics

Supervised learning models were evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

Clustering performance and patterns were analyzed using visualization and cluster interpretation techniques.

---

## 🔍 Key Insights

- Countries with higher education and healthcare investment generally showed higher life expectancy.
- Economic and health-related indicators strongly influenced prediction performance.
- Clustering revealed distinct groups of countries based on development and healthcare metrics.
